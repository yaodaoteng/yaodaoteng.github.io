hexo+GitHub搭建

前提：git和node.js安装完毕

一、安装hexo

  1. 利用npm命令安装。在任意位置打开Git bash

  2. 输入 npm install -g hexo-cli

    二、初始化hexo

    1. 创建文件夹（最好在一个文件盘）
    2. 在hexo文件夹中，右键运行Git bash，输入hexo init

    初始化成功后的一类文件

三、写文章

  • 在Hexo\source_posts文件下，新建.md文件就可以写文章
  • 生成静态文件 hexo g
  • 启动服务器 hexo s 默认情况下，访问网址为： http://localhost:4000/。

四、配置

在 _config.yml 中修改大部分的配置。

四、部署到GitHub上

  1. 申请GitHub账号
  2. 创建new repository （注意Repository name一定得是http://yourname.github.io，这样才能使用这个地址访问到你的Github page，填好Repository name，点击Create repository。）
  3. 打开hexo文件，git init初始化Git
  4. 打开hexo文件下，找到在_config.yml进行配置

  1. 安装hexo-deployer-git自动部署发布工具

    npm instal lhexo-deployer-git –save

    （在Hexo文件夹下 Git Bash）

  2. 输入如下命令：hexo clean && hexo g && hexo d

  1. 测试访问

在浏览器输入：https://yaodaoteng.github.io/
